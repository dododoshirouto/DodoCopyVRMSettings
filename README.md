# DodoCopyVRMSettings

## 概要

VRMMetaなどの設定をコピーするスクリプト

他に、VRMFirstPerson、VRMLookAtHead、VRMSpringBone、VRMSpringBoneColliderGroupの設定もコピーします

## 使い方

1. 新しいEmptyObjectを作成し、CopyVRMSettingsをアタッチする
2. Sourceに元となる設定済みVRMオブジェクトをいれる
3. Targetsにコピー先となる未設定のVRMオブジェクトをいれる
4. コンポーネントのコンテキストメニューから「Copy」を選ぶ
